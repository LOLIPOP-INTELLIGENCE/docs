## How to Connect

<Steps>
  <Step title="Open Settings">
    Go to your Kairos dashboard and click on Integrations
  </Step>
  
  <Step title="Choose Tool">
    Select the tool you want to connect from the list
  </Step>
  
  <Step title="Authorize Access">
    Click "Connect" and follow the authorization flow
  </Step>
  
  <Step title="Start Using">
    That's it! Kairos can now work with your tool
  </Step>
</Steps>

## Integration Features

### Secure Authentication
- OAuth 2.0 for secure connections
- Granular permissions — only access what's needed
- Revoke access anytime from settings
- No passwords stored

### Smart Sync
- Real-time data access
- Automatic refresh of connections
- Handles rate limits intelligently
- Offline fallback options

### Cross-Tool Workflows
Connect multiple tools for powerful automations:

<Example>
**Email → CRM → Slack**
"When I get an email from a new lead, add them to Salesforce and notify the sales team in Slack"

**Calendar → Task Manager → Email**
"Check my calendar, create tasks for meeting prep, and email attendees with agendas"

**Sheets → Docs → Email**
"Analyze this month's data, create a report in Docs, and email to stakeholders"
</Example>
